    MIS PROYECTOS

    Aplicación PyQt5

    GUI Moderna en Python con PyQt5 - Menú Lateral CRUD con SQLite

    En este video se realiza una interfaz grafica en el lenguaje de programación Python usando la librería PyQt5, esta aplicación consta de un menú desplegable en la parte izquierda de donde podemos acceder a las distintas paginas creadas, en estas paginas creamos un CRUD usando SQLite la cual nos permite leer, registrar, modificar y eliminar datos.

    Aplicación Tkinter

    Interfaz Gráfica para Control de Arduino con Python y Tkinter

    En este emocionante proyecto, se presenta una interfaz gráfica desarrollada en Python utilizando la librería Tkinter para interactuar con un Arduino y enviar y recibir datos analógicos. La interfaz permite controlar señales analógicas y visualizarlas en tiempo real utilizando Matplotlib. Además, ofrece la posibilidad de variar las salidas PWM mediante deslizadores (Sliders) o escalas (Scale), proporcionando una experiencia de usuario interactiva y personalizable.

    Aplicación Kivy

    Aplicación del Clima con KivyMD y la API de OpenWeather

    Esta aplicación del clima desarrollada en KivyMD ofrece una experiencia intuitiva y atractiva para usuarios que deseen acceder fácilmente a la información meteorológica de cualquier ubicación. Integrando la potencia de la API de OpenWeather, proporciona datos precisos y actualizados sobre condiciones climáticas en tiempo real.

    Aplicación Flet

    Sistema CRUD en Python con Flet y SQLite

    En este proyecto completo, se detalla el desarrollo de un Sistema CRUD (Crear, Leer, Actualizar, Borrar) en Python, implementando la biblioteca Flet junto con SQLite. Este sistema permite la gestión eficiente de datos a través de un formulario responsive que incluye campos esenciales como nombre, edad, correo y teléfono.
